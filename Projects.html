CS648 (Randomized Algorithms)

Randomized Algorithm for Hashing in Linear Space and O (1) worst case search time [Results_1] [Results_2]

  • Implemented the linear space and O(1) search time randomized hashing algorithm for a 2-universal hash family using linked list.
  • Compared the time in finding the perfect hash functions for different scaling constants at each level of hashing.
  • Verified that the time taken to find the perfect hash function is O(1) with high probability.
  • Showed that increase in scaling constant in the size of hash table upto a threshold of 1 decreased the time in finding perfect hash function.

EE604 (Image Processing)

Block Compressed Sensing (CS) of Natural Images[Report]

  • Implemented a 2- step block- by-block CS non-linear image reconstruction algorithm with low computational complexity
  • Divided original image into small blocks and sampled each block using the same measurement matrix
  • Analyzed the trade-off between computation time and reconstruction error for different block size
  • Optimum block size of 32 for a 256 x 256 image was obtained

Microwave Lab IIT Kanpur

Application of Genetic Algorithms for the design of Metamaterial Band Pass Filters (SURGE IITK) [Report]

The objective of this project was to understand the application of Genetic Algorithm in the optimisation of design of CRLH Transmission Line (TL) Metamaterials. Metamaterials are a class of materials which exhibit properties not found in naturally occurring materials. Left Handed Metamaterials(LHMs) have negative permittivity and permeability. The basic unit of a metamaterial is known as a unit cell. LHMs can be realised by a CRLH TL. For a metamaterial to be used for a specific application, its unit cell has to be designed and optimized to get the desired performance. It has been found that using metamaterial zeroth order resonators (ZORs) one can realise narrow microwave band pass filters with size reduction of about 60% as compared to conventional RH filters. Genetic Algorithms are used for optimization purpose. These algorithms work their principle from natural genetics. The advantage of using GA is that a GA search converges to a global maxima rather than other gradient based classical optimisation which converge to local maxima. In this work, GAs were used for two applications, (i) to optimize the design of unit cell to make it balanced and (ii) to optimise the design of metamaterial band pass filter to minimise the insertion loss.
